Events

Es gibt zwei wege für den eventListener: .addEventListener(); und den parameter .on...;

Window

Window.onload;

Dieses event führt einen callback aus, wenn das browserfenster fertig geladen hat. Also wenn alle resourcen der angezeigten html datei heruntergeladen wurden.

  • window.addEventListener('load', (event) => {});
  • window.onload = (event) => {}
weitere .on... events:
  • element.onclick = (event) => {}
  • element.onmouseover = (event) => {}
  • element.onmouseout = (event) => {}
  • element.onmousedown = (event) => {}
  • element.onmouseup = (event) => {}
  • element.onmousemove = (event) => {}
  • element.onkeydown = (event) => {}
  • element.onkeyup = (event) => {}
  • element.onfocus = (event) => {}
  • element.onsubmit = (event) => {}
  • element.onblur = (event) => {}
  • element.onchange = (event) => {}
  • element.oninput = (event) => {}
  • window.onload = (event) => {}
  • window.unload = (event) => {}
  • window.resize = (event) => {}

um die kurzen event befehle zu nutzen, müssen wir uns eigentlich nur merken, das wir dem event des .addEventListeners(); ein on voransetzen, also zum beispiel wird aus .addEventListener('click', () => {}) - .onclick = () => {};

In react nutzen wir überwiegend diese kurzform, dort wird allerdings nach dem on der erste buchstabe groß geschrieben, aus onchange, wird also zum beispiel onChange.
