Variablen

variablen werden in SASS im gegensatz zu css nicht in das :root element geschrieben, sondern können sich überall im code befinden. Der typische weg ist, sie am anfang einer scss datei zu deklarieren, oder in einer externen datei, aus der man sie dann importiert. Allgemein gilt bei SASS der ansatz, das man verschiedene arten von inhalten, in verschiedenen dateien trennt, um sie besser wartbar zu machen. Für unsere testzwecke schreiben wir allerdings fürs erste alles in eine datei.

Wir wechseln in unsere scss datei und legen ein paar variablen an.

Verschachteln von selektoren

in SASS haben wir die möglichkeit selektoren strukturierter und ordentlicher zu sortierne, indem wir zusammengehörende inhalte miteinander verschachteln. So können wir einen selektorenbaum ähnlich der HTML struktur der jeweiligen elemente anlegen, und weitere dinge wie hover, nth-child oder unterklassen direkt mit dem selektor zusammenfügen.
