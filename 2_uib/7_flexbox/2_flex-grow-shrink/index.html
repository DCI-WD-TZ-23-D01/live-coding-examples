Flexbox

flex-grow: ;

mit flex-grow: ; können wir definieren, wieviel platz ein element innerhalb des elternelementes einnimt, ohne pixelangabe, oder ähnliche größenangabe, sondern in anteilen des platzes, den es hat.

der default wert von flex-grow: ; ist 0, das bedeutet, das die elemente nur soviel platz einnehmen, wie sie brauchen, um ihren inhalt anzuzeigen.

flex-grow: 0;

flex-shrink: ;

im gegenteil zu flex-grow: ; können wir mit flex-shrink: ; definieren, wie sich ein element verhält, wenn es von der größe her nicht mehr in den container passen würde, und verkleinert werden müsste, um den vorgesehenen platz zu nutzen.

WICHTIG: Der default wert von flex-shrink: ; ist 1, wodurch elemente immer nur so groß sind, das sie für das elternelement verkleinert werden.
