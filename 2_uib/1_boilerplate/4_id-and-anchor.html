Element ID

Um verschiedene elemente zu identifizieren und direkt ansprechen zu können, gibt es sogenannte "id's", "Identifikatoren". Mithilfe von id's können wir später in CSS elemente direkt stylen, oder in JavaScript direkt ansprechen, um sie mit inhalten oder stylings zu füllen.

Id's haben allerdings noch eine weitere funktion. Mithilfe der anchor elemente können wir direkt an positionen in einer HTML datei springen, und so eine interaktive möglichkeit schaffen um sich auf unserer seite zu bewegen. Damit wir sagen können WO wir hinspringen wollen, vergeben wir an unsere ankerziele ID's.
