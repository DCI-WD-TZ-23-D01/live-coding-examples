text-decoration: ;

Die css eigenschaft text-decoration: ; ermöglicht webentwicklern, texte zu unterstreichen, durchzustreichen oder auch einen strich über den text zu setzen, was sowohl die ästhetik als auch die usability von webseiten verbessert.

Durch text-decoration: ; ist es möglich webseiten ohne aufwendige grafiken oder scripte visuell ansprechend zu gestalten.

Kein strich

Der standardwert von text-decoration: ; ist "none", es wird also kein strich angezeigt. So wird der text einfach dargestellt und fällt nicht weiter auf.

Unterstrichener text

Die meist benutzte einstellung für text-decoration: ; ist text-decoration: underline;. Bei dieser einstellung bekommt der text einen strich unterhalb seines textinhaltes. Dies ist zum beispiel der standardwert bei einem "a" element.

Dieser text ist unterstrichen

Wie wir sehen ist der text jetzt auffälliger, er zieht unsere blicke direkt auf sich.

Wir können jedem element text-decoration: ; geben, die option wird dann auf text innerhalb des elementes angewandt, ist also vererbbar.

Durchgestrichener Text

Eine gern benutzte einstellung ist "line-through", hier wird der text durchgestrichen, was zum beispiel gut ist um gelöschte inhalte oder vergriffene artikel darzustellen.

text-decoration befehle kombinieren

Das interessante am text-decoration: ; property ist, das es möglich ist, mehrere angaben miteinander zu kombinieren. So sind wir in der lage, einem text zum beispiel einen oberstrich und einen unterstrich zur gleichen zeit zu geben.

Anpassung des striches

text-decoration: ; besteht aus mehreren untereigenschaften, mit denen sich das aussehen des striches einstellen lässt. Diese optionen erlauben uns, fortgeschrittene visuelle effekte zu erzielen, indem wir farbe, dicke und stil des striches anpassen.

Es gibt folgende unteroptionen

  • text-decoration-line: ; = gibt an, welchen strich wir haben wollen
  • text-decoration-style: ; = gibt an welche art strich wir wollen, hier gelten die optionen: solid, double, dotted, dashed und wavy
  • text-decoration-color: ; = gibt an welche farbe der strich haben soll
  • text-decoration-thickness: ; = gibt an wie dick der strich sein soll

Durch diese optionen können wir den strich stylen wie wir wollen, zum beispiel können wir den strich nachbilden, den schreibprogramme darstellen, wenn sie ein wort nicht erkennen.
