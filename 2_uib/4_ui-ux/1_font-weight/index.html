Font-Weight (schrift dicke)

Um text auf webseiten hervorzuheben gibt es verschiedene möglichkeiten. Eine dieser möglichkeiten ist es, den text dicker erscheinen zu lassen, für die definition dieser dicke, nutzt man beim web design den begriff "gewicht" oder "weight". Das gewicht gibt an, wie viel aufmerksamkeit ein text auf sich ziehen soll.

HTML elemente

obwohl man jedem html element die fähigkeit geben kann fetten text zu beinhalten, gibt es grundsätzlich 2 semantische html elemente, die für das "fettschreiben" von text zuständig sind. Sie sind derivate von "span", und sind somit also inline elemente. Diese elemente kann man für verschiedene zwecke nutzen.

Beide elemente "strong" und "b", dessen name die abkürzung für "bold", also "fett" ist, haben die selben einstellungen, werden aber aus semantischen gründen für verschiedene zwecke benutzt.

das b element

das "b" element wird hauptsächlich genutzt um inhalte innerhalb eines paragraphen fett darzustellen. Wie im folgenden text:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Maxime itaque molestias esse, optio ipsam unde enim dolor similique alias necessitatibus cum? Similique dolores eligendi officiis sed, reiciendis adipisci alias excepturi!

das strong element

Das strong element wird im gegensatz zum b element dafür benutzt, ganze text passagen, wie zum beispiel warnhinweise, fett anzuzeigen, etwa so:

Dies ist ein wichtiger hinweis!

Der CSS befehl font-weight: ;

Durch CSS haben wir die möglichkeit, jegliche elemente die fähigkeit zu geben, fetten text zu beinhalten. Wir haben ausserdem im gegensatz zu den semantischen elementen, hier die möglichkeit anzugeben, wie fett ein text sein soll.

Der defaultwert für font-weight: ; ist "normal", dies stellt den text in einer normalen, gut lesbaren dicke dar.

Ein normaler text

Wenn wir font-weight: ; auf "bold" setzen, bekommen wir die gleiche einstellung wie bei b und strong.

Ein fetter text

Um spezifische anpassungen zu machen, haben wir die möglichkeit mit zahlenwerten das gewicht anzugeben. Die möglichen zahlen gehen von 1 bis 1000, wobei 400 dem normalwert entspricht, und man in 100er schritten arbeitet. Sollte man ein gewicht haben, das die gewählte schriftart nicht unterstützt, sucht der browser automaitsch die nächst unterstützte dicke selbst aus.

Da grundsätzlich immer diese 100er schritte benutzt werden, haben sie eigene namen:

100 (thin)

200 (extra light)

300 (light)

400 (normal/regular)

500 (medium)

600 (semi bold)

700 (bold)

800 (extra bold)

900 (black/heavy)

950 (ultra black)

Wenn man also mal hört "setze die font auf medium", bedeutet dies, ein font-weight: ; von 500.

Das schnelle dicker oder dünner machen von textinhalten

Wenn ein element über eine gewichtseinstellung verfügt, und man aber innerhalb des elementes etwas text braucht der vielleicht dicker oder dünner ist, als der angegebene, kann man ihn, da das gewicht vererbt wird, schnell anpassen.

ein dünnerer text

ein dickerer text

Wir sind in der lage den text dünner zu machen, als der vordefinierte normalwert, indem wir font-weight: ; auf "lighter" setzen. Das gegenteil, also dickeren text erhalten wir, wenn wir font-weight: ; auf "bolder" setzen.
