Selektoren

In CSS nutzen wir selektoren um das element, das wir stylen/ansprechen wollen auszuwählen. Ein CSS kommando ist wie folgt aufgebaut:

  • [element/name/klasse/id] { = der selektor
  • [eigenschaft]: [wert]; = die option
  • } = das ende des selektors

Mit komma getrennt, können wir mehrere selektoren auswählen und ihnen die selben optionen geben.
