Typische benennung von Dateien

HTML

Die haupt HTML datei eines projektes heißt immer "index.html". Eine index datei wird in linux systemen immer direkt anstelle des ordner-inhaltes aufgerufen, das bedeutet, wenn wir beispielsweise an folgender stelle auf der festplatte eine index-datei hätten: ./test/seite/index.html und wir bewegen uns nach ./test/seite im browser, würde automatisch die index.html aufgerufen werden.

CSS

Die haupt CSS dateo, in der entweder alle styles liegen, oder in der die verschiedenen styles zusammengeführt werden heißt immer style.css. Wenn verschiedene dateien zusammengefügt werden, können wir sie auch index.css nennen, das ist aber eher in SASS der fall.

Das Externe stylesheet

Ein externes stylesheet bietete mehrere vorteile und gilt oft als die beste möglichkeit des stylings im vergleich zu internen (internal styles) und inline-styles.

Vorteile eines Externen Stylesheets

Trennung von Inhalt und Darstellung

Externe stylesheets trennen den HTML-Code von den styles, was die klare unterscheidung von Inhalt und Darstellung ermögicht und somit die wartung und lesbartkeit des codes verbessert. Im gegensatz dazu sind interne und inline styles direkt im HTML dokument enthalten, was zu einer unübersichtlichen und schwer wartbaren codebasis führen kann, besonders bei großen projekten.

Wiederverwendbarkeit

Die verwendung eines externne stylesheets ermöglicht die wiederverwendung von CSS-inhalten auf mehreren dateien. Wodurch eine konsistente gestaltung im gesammten projekt erreicht wird. Im gegensatz dazu gelten interne und inhline styles nur für die jeweilige seite oder das spezifische element, was die wiederverwendung erschwert, und bei mehreren dateien tatsächlich unmöglich macht.

Cache-Nutzung

Dee nutzung eines externen stylesheets ermöglicht es Browsern, die dazugehörigen CSS-Dateien zwischenzuspeichern (cachen), was die ladezeit verbessert, da die datei bei wiederholten besuchen oder wiederholter verwendung nicht erneut heruntergeladen werden muss.

Nutzung des Stylesheets

Zu allererst erstellen wir eine datei namens style.css und schreiben dort ein paar css definitionen hinein, damit wir die änderungen gleich sehen können, sobald wir den link zur CSS datei in der HTML datei hinterlegt haben.

Um das externe stylesheet in unserer HTML datei zu nutzen, müssen wir sie im HEAD der html datei verlinken. Dafür haben wir das element "link", das wir im head platzieren können.
