CSS Intro (Styles)

Um webseiten ein design zu geben, nutzt man CSS, was für Cascading Style Sheet steht, also Gestufte Stilvorlagen. Kaskadierend bedeutet, das ein dokument nicht nur von einem einzelne stylesheet formatiert werden kann, sondern von einer vielzahl von styles, die aus verschiedenen quellen stammen können und aufeinander aufbauen können.

Mit CSS können wir farben, abstände, animationen und positionen für einzelne, oder mehrere elemente definieren und so ein stimmiges, zusammenpassendes und professionelles aussehen oder design für eine seite erschaffen.

Stylesheets

CSS befehle definieren wir in sogenannten stylesheets, das sind sammlung von selektoren und befehlen, die aussagen wie der style von verschiedenen elementen auf der webseite aussehen soll.

Es gibt grundsätzlich 3 verschiedene arten (oder positionen) wie man CSS inhalte definiert.

  • Inline Styles: hier schreiben wir die CSS styles direkt in das element
  • Internal Styles: hier schreiben wir die css styles in den header .html datei
  • External Stylesheet: hier schreiben wir unsere css styles in eine externe .css datei, und verlinken diese in unserer html datei

Inline Styles

Inline styles können wir nutzen um schnell und direkt einzelne elemente zu stylen. Der nachteil ist, das wir für jedes element einzeln css styles festlegen müssen, und sie nicht dynamisch wiederverwenden können. DIese technik lernen als erstes, um uns mit css vertraut zu machen.

Internal Styles

Bei dieser art des stylings wird im head der HTML datei ein "style" element erstellt, in das man die CSS inhalte hineinschreibt, man es sehr dynamisch verwenden und hat den vorteil, das sich alle inhalte die mit em layout er seite zutun haben, also sowohl css wie auch html in einer datei befinden. Der nachteil ist, das, wenn wir mehrere html dateien haben, wir auf den style nicht in einer anderen datei zugreifen können.

External Stylesheets

Eine etwas ordentlichere herangehensweise ist das externe stylesheet, hier wird eine einzelne oder mehrere dateien des types .css angelegt, in der die CSS stylings hinterlegt wurden. Dies ist in produktion beliebt. da man so bestimmte inhalte und elemente in einzelnen dateien getrennt pflegen kann, und eine strikte trennung von inhalt und design hat. Außerdem können wir die jeweiligen stylesheets miteinander verknüpfen, und in jeder .html datei die wir nutzen anwenden.

Aufbau eines CSS Befehls

Ein CSS befehl besteht aus einem selektor, und mehreren deklarationen. Der selektor wählt das element aus, das gestyled werden soll, während die deklaration die stileingeschaft und ihre werte festlegt. Ein Beispiel

p { color: red; font-size: 16px; }

Das beispiel zeigt das typische aussehen eines CSS befehls

  • Wir sagen "was wir ansprechen", hier das "p" element
  • Wir öffnen den befehl mit einer geschwungenen klammer (ALT GR + 7)
  • Wir fügen die gewünschten optionen ein, bei der wir jede mit einem ; abschließen
  • Am ende schließen wir unseren befehl wieder mit einer geschwungenen klammer (ALT GR + 0)

In unserem code beispiel sagen wir, das alle "p" elemente folgendes styling haben sollen:

  • color (farbe): red (rot)
  • font-size (schriftgröße): 16px (16 pixel)

Bei inline styles benötigen wir keinen selektor, weil wir direkt auf dem jeweiligen element unsere styles hinzufügen, indem wir das attribut "style" verwenden.
