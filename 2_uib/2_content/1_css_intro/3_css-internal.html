Spezifische gruppen von elementen verändern

Eine weitere art der selektion ist die klasse. Eine klasse beinhaltet einen deklarierten namen, der im gegensatz zu id's mehrfach verwendet werden kann, und so mehrere elemente mit den gleichen eigenschaften versehen kann.

Eine klasse wird definiert, indem die jeweiligen elemente das attribut "class" bekommen. In CSS werden klassen mit einem . von dem namen angesprochen
