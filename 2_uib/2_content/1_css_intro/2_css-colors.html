CSS Farben

Der hauptteil des stylings von webseiten besten, neben dem bestimmen von abständen oder größen verschiedener elemente vor allem darin, zu definieren, welche farben die verschiedenen elemente haben, um zusammen dem benutzer (der user) der seite eine professionelle, angenehme und ansehnliche erfahrung zu gewährleisten.

Farbarten in CSS

In CSS gibt es verschiende arten von Farbwerten, die definieren wie eine farbe zusammengestellt wird, darunter benannte fatben, hexidezimalfarben (hexcolors) und RGB/RGBA. Jede dieser arten bietet unterschiedliche möglichkeiten, farben zu definieren und anzuwenden.

Benannte Farben

Benannte farben sind vordefinierte farbwerte, die leicht durch ihren englischen namen aufgerufen werden können. Zum beispiel: rot, grün und blau.

RGB und RGBA

RGB (Rot, Grün und Blau) und RGBA (Rot, Grün, Blau mit dem sogennanten alpha kanal, für transparenz) werte bieten eine detaillierte kontrolle über farben. Zum beispiel: ein helles blau

Der wert für jede farbe geht von 0 für "nichts" bis 255 für "voll". So kann man farben fantastisch mischen und anteile verschidener farben miteinander kombinieren.

Beispiele für RGB
  • Schwarz
  • Rot
  • Blau
  • Beere
  • Türkis

Durch das mischen von 255*255*255 farben ergibt sich eine gesamtmenge an von 16.7 millionen möglichen farben.

Hexadezimalfarben

In CSS werden farben meist durch hexadezimalwerte dargestellt. Das hexadezimalsystem basiert auf 16 zeichen, von 0 bis 9 und von A bis F.
Diese werden in einer kombination von 6 zeichen verwendet, wobei jeweils zwei für die Rot-, Grün- und Blauanteile stehn. Zum beispiel: #ff0000 repräsentiert rot.

Jedes zeichen repräsentiert 4 bits, daher besteht ein hexadezimalwert aus einer kombination von zwei zeichen, was 8 bit oder ein byte ergibt.

Aufbau eines Hexadezimalwerts

Wie oben erwähnt besteht ein hexadezimalwert für eine farbe aus sechs bis acht zeichen. Diese zeichen sind in gruppen unterteilt:

  • Erstes Zeichenpaar: Der rotanteil der farbe
  • Zweites Zeichenpaar: Der grünanteil der farbe
  • Drittes Zeichenpaar: Der blauanteil der farbe
  • (OPTIONAL) Viertes Zeichenpaar: Der transparenzanzeil der farbe

Jedes paar dieser zeichen repräsentiert einen wert zwischen 00 und FF im hexadezimalsystem, was auch hier einen bereich von garnicht, bis absolut (0 bis 100%) entspricht. Hier sind einige beispiele:

  • #FF0000: Vollständige farbe rot (255, 0, 0 in RGB)
  • #00FF00: Vollständige farbe grün (0, 255, 0 in RGB)
  • #0000FF: Volständige farbe blau (0, 0, 255 in RGB)
  • #FFFFFF: Vollständige weiße farbe (255, 255, 255 in RGB)
  • #000000: Vollständiges Schwarz (0, 0, 0 in RGB)
  • #3CAFE2: Kaffeeblau (60, 175, 226 in RGB)

Eine kurzschreibweise für farbwerte, in denen sich die jeweiligen zeichen in der gruppe wiederholen kan verwendet werden indem man jeweils nur das erste zeichen der gruppe nutzt. Aus #AADDAACC wird dann zum beispiel #ADAC.

Anwendung von Farben in CSS

Gründsätzlich gibt es 2 verschiedene farbebenen mit denen wir arbeiten können. Die textfarbe und die hintergrundfarbe.

Textfarbe (color)

Mit der textfarbe können wir definieren, welche farbe ein bestimmter text haben soll, wie schon mehrfach in den vorherigen zeilen gesehen, nutzen wir dafür die eigentschaft "color" und übergeben ihr benannte, hex oder rgb farben. Ein beispiel:

Lorem ipsum, dolor sit amet consectetur adipisicing elit. Enim magnam rerum, vel repudiandae quas temporibus atque placeat vero illum amet ipsum aut, libero aperiam laboriosam quasi voluptate quia soluta quos.

Hintergrundfarbe (background-color oder background)

Mit der hintergrundfarbe können wir definieren welche farbe der hintergrund eines elements haben soll. Hierfür nutzen wir die eigenschaft "background-color", oder die kurzform (die sogenannte shorthand property) "background".
