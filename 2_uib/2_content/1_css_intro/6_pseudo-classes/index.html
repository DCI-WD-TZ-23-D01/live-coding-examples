Pseudo-Klassen

Pseudoklassen sind spezielle selektoren, die dynamische zustände von HTML-elementen ansprechen. Zum beispiel wird die ":hover" pseudo-klasse verwendet, um styles auf eine element anzuwenden, während es vom mauszeiger berührt wird. Diese klassen bieten eine möglichkeit, das verhalten von elementen in verschiedenen interaktionszuständen zu stuern und damit die benutzererfahrung zu verbessern.

:hover

Diese pseudo-klasse wird verwendet um für den fall styles zu definieren, in dem die maus sich über einem element befindet. Dies wird oft für interaktive elemente verwendet um visuelles feedback zu geben.

:active

Diese pseudo-klasse wird angewendet, wenn eine element durch einen klick aktiviert wird. Sie ist nützlich um anzuzeigen das in diesem moment eine aktion ausgeführt wird.

Google link

:visited

Diese pseudo-klasse wird verwendet um schon mal besuchte, von nicht besuchten links zu unterscheiden. Die unterscheidung kann durch zurücksetzen der browserdaten abgebrochen werden.

Benutzer und Beispiele

pseudo-klassen werden definiert indem sie an einen selektor angefügt werden. Sie starten immer mit einem doppelpunkt (:) und ihre namen sind vordefiniert.

In produkton vergeben wir bei links oft selektoren gruppen, um besuchte links oder aktive links an standarddesigns anzupassen

Google Link

Elemente

Im gegensatz zu :visited funktionieren :hover und :active nicht nur bei links, sondern bei allen elementen, so können wir verschiedenen elementen die möglichkeit geben auf die interaktion des benutzers zu reagieren.
