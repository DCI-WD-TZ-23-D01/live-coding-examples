Größeneinheiten

Um elemente auf einer webseite eine größe zu geben, egal ob höhe oder breite nutzen wir den platz den wir im browserfenster zur verfügung haben so gut wie möglich aus. Um dies zu tun haben wir verschiedene möglichkeiten größen zu definieren.

Als grundwert für die größeneinheiten nutzen wir den sogenannten Pixel, der durch die bildschirmauflösung des anzeigegerätes, wie zum beispiel dem Monitor oder dem smartphone display angegeben wird. Es gibt aber abgesehen von der reinen pixel-angabe noch andere werte, die sich relativ zur pixelauflösung verhalten.

Pixel (px)

Der Pixel (px) ist eine absolute größeneinheit und definiert die größe eines elements in festen pixelwerten. Zum beispiel ist dieser text standardmäßig 16 pixel groß. Das bedeutet das er auf jedem monitor die selbe pixelgröße hat, bei verschiedenen auflösungen ist er für den benutzer also verschieden groß.

rem

rem ist eine relative größeneinheit, die die größe relativ zum Root-Element (html) verändert, das bedeutet, wenn wir die größe von REM dokumentenweit anpassen, wird basierend darauf alles was in rem angegeben wurde, abhängig von diesen angaben verändert.

  • 1rem = 16px (wenn nicht anders definiert)
  • 2rem = 32px
  • 0.5rem = 8px
