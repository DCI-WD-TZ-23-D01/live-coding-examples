CSS-Grid

Grid Template Areas (vordefinierte flächen)

der befehl grid-template-areas: ; ist eine erweiterte funktion des css-grid layouts, das es uns ermöglicht, komplexe layouts auf einfache und lesbare weise zu definieren. Durch die verwendung von benannten bereichen können wir den container in abschnitte unterteilen und diesen abschnitten dann spezifische namen zuweisen, um die platzierung von elementen im raster (grid) zu steuern. Diese namen werden dann verwendet, um die platzierung von grid elementen in css zu definieren, wodurch der code übersichtlicher und leichter zu warten ist.
