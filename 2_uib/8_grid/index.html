CSS-Grid

Um layouts besser gestalten zu können und eine präzise kontrolle über die anordnung von elementen auf webseiten zu ermöglichen, wurde 2017 von der W3C das fortschrittliche layout-system CSS-Grid entwickelt. Im vergleich zu älteren layout methoden bietet CSS-Grid eine zweidimensionale rasterstruktur, die es entwicklern erlaubt, elemente sowohl horizontal, wie auch vertikal genau zu positionieren. Von komplexen seitenlayouts wie magazinseiten, bis hin zu dashboards, tabellen oder bildergallerien bietet CSS-Grid eine vielzahl von anwendungsmöglichkeiten und ermöglicht eine verbesserte responsivität für verschiedene bildschirmgrößen.

Nutzung von CSS-Grid

Die fr Einheit

Um innerhalb eines grids verschiedene elementen dynamisch gleiche größen geben zu können, gibt es für grid elemente eine eigene größeneinheit, diese einheit nennt man "fr", für "fraction". Ein fr ist also ein definierter anteil (eine fraktion) des verfügbaren platzen.

display: grid;

mit display: grid; setzen wir den container in den grid modus, damit bekommen auch alle kinder die display option "grid".

für unsere beispiele nutzen wir einen container mit 12 kindern, die wir durch css grid frei platzieren können.

Spalten

mit grid-template-columns: ; legen wir eine vorlage für die horzontalen spalten des grid netzen an.

im folgenden beispiel erstellen wir unseren 12 kinder container in dem die elemente so angelegt werden sollen, das automatisch die inhalte in 3 elemente pro zeile eingeteilt werden, und sich wiederholen, bis alle boxen platziert wurden.
