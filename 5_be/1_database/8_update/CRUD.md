# Mongoose + CRUD

(C)reate:

- `save`
- `create`
- `insertMany`

(R)ead:

- `find`
- `findOne`
- `findById`

(U)pdate:

- `updateOne`
- `updateMany`
- `findByIdAndUpdate`
- `save`, doch das Dokument muss direkt aktualisiert werden

(D)elete:

- `deleteOne`
- `deleteMany`
- `drop`
  - Bsp.: `Organizer.collection.drop()`
- `findByIdAndDelete`
